# BMI Calculator (Python)

A simple Python project that calculates Body Mass Index (BMI) based on height and weight.

## Features
- Takes user input for height and weight
- Calculates BMI using formula
- Shows BMI category:
  - Underweight
  - Normal
  - Overweight
  - Obese

## Learning Outcome
- Python input handling
- Conditional statements
- Basic arithmetic operations
